Tentu, ini adalah program HTML lengkap yang dirancang untuk berinteraksi dengan semua fitur di file app.py yang baru Anda berikan. ## Fitur Utama Tampilan Ini * Antarmuka Dua Kolom: Di sebelah kiri adalah panel chat untuk percakapan, dan di sebelah kanan adalah panel informasi untuk menampilkan hasil rekomendasi. * Input User ID Dinamis: Anda bisa memasukkan user_id apapun untuk memulai sesi, memudahkan pengujian sebagai talent atau company. * Tombol Aksi: Terdapat tombol "Mulai Sesi", "Info Rekrutmen", dan "Reset Chat" yang terhubung langsung ke endpoint API yang sesuai. * Panel Informasi Interaktif: Panel di sebelah kanan akan menampilkan: * Profil singkat talent yang sedang dianalisis. * Pesan outreach yang sudah diformat. * Hasil peringkat lowongan dari model AI, lengkap dengan skor dan alasan. * Daftar lowongan pekerjaan terbaru. * Desain Responsif: Tampilan akan otomatis menyesuaikan menjadi satu kolom di perangkat mobile. ## Kode Lengkap index.html Salin semua kode di bawah ini dan simpan ke dalam file templates/index.html.
